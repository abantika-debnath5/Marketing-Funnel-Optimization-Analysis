# 📊 Marketing Funnel Optimization Analysis

## 📌 Project Overview
This project analyzes a multi-stage marketing funnel to identify conversion bottlenecks and optimize user journey performance. The analysis is performed using *Python and Power BI*, ensuring consistency across tools and providing actionable business insights.

---

## 🎯 Objective
To answer the key business question:

> *Which stage of the marketing funnel contributes most to revenue loss, and what actions can improve conversion rates?*

---

## 🧠 Funnel Stages
The analysis is based on a user-level funnel:

- *Visit (Awareness)*
- *Product View (Interest)*
- *Add to Cart (Intent)*
- *Purchase (Conversion)*

---

## 📂 Dataset
The dataset consists of user session-level data with the following key features:

- User_ID
- Session_ID
- Time_stamp
- Page_Type
- Device_Type
- Referral_Source
- Items_In_Cart
- Purchased

---

## ⚙️ Tools & Technologies

- *Python (Pandas)* → Data cleaning & primary data analysis  
- *Power BI* → Dashboard visualization & storytelling  

---

## 🔍 Key Analysis Performed

### 1. Funnel Analysis
- Built a user-level funnel using binary flags
- Calculated stage-wise conversion rates
- Identified major drop-off points

---

### 2. Device Segmentation
- Compared performance across:
  - Desktop
  - Mobile
  - Tablet
- Identified device-specific conversion inefficiencies

---

### 3. Referral Source Analysis
- Evaluated traffic quality across channels:
  - Direct
  - Google
  - Email
  - Social Media
- Measured conversion effectiveness per source

---

## 📊 Key Insights

- 🚨 *48.8% drop-off occurs at the Cart → Purchase stage*, indicating a major conversion bottleneck  
- 📱 *Tablet users show the lowest conversion rate*, suggesting UX or performance issues  
- 📢 *Social Media drives traffic but underperforms in conversion*, indicating low-intent users  
- 🔍 *Direct and Google channels show higher conversion efficiency*

---

## 💡 Business Recommendations

- Optimize *checkout experience* to reduce abandonment  
- Improve *tablet responsiveness and UI*  
- Reallocate budget toward *high-converting channels*  
- Refine *targeting strategy for Social Media campaigns*
